# UTILS

*Répertoire contenant des notes, documentations ou autres éléments ne nécessitant pas de répertoire spécifique*

```
UTILS/
├── SLR_PXE/
│   └── README.md          # Guide complet pour configurer PXE sur un serveur Linux en routeur
├── serveurDHCP/
│   └── installation.md    # Instructions pour mettre en place un serveur DHCP dans une VM Proxmox
├── switch/
│   └── hp/
│       └── setupSshOnSwitch.md   # Tutoriel pour activer et configurer SSH sur un switch HP 3600
```

---

Chaque dossier contient une documentation ou des notes détaillées sur le sujet indiqué :  

- **SLR_PXE/README.md** : décrit la configuration PXE sur un serveur Linux en mode routeur.  
- **serveurDHCP/installation.md** : explique comment installer et configurer un serveur DHCP sur une VM Proxmox.  
- **switch/hp/setupSshOnSwitch.md** : fournit les étapes détaillées pour activer SSH, créer un compte admin et se connecter sur un switch HP 3600 (Comware v5).
